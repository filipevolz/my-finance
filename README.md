# my-finance

Sistema de gerenciamento financeiro pessoal desenvolvido com React (frontend) e NestJS (backend).

## Tecnologias

- **Frontend**: React, TypeScript, Vite, Styled Components, Shadcn/ui, Lucide React
- **Backend**: NestJS, TypeScript, PostgreSQL, TypeORM
- **Autenticação**: JWT
- **Validação**: Zod (frontend), class-validator (backend)

## Funcionalidades

- Gestão de receitas e despesas
- Categorização de transações
- Dashboard com estatísticas e gráficos
- Filtros por período
- Últimas transações
- Análise de despesas por categoria
